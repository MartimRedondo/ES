## Introdução
Este projeto implementa um **Cofre Digital**, um sistema seguro onde utilizadores podem armazenar e gerir ficheiros com garantias robustas de segurança. O sistema opera através de uma arquitetura cliente-servidor, onde cada utilizador interage com o sistema através de uma aplicação cliente (CLI) que se conecta a um servidor central. O servidor gerencia o armazenamento e as políticas de acesso, enquanto garante o isolamento e proteção dos dados.
