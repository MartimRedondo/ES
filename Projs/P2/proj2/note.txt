SIMULAÇÃO:

(opcional)
    - python client_cli.py sessions         - verifica quais sessões estão ativas, porque caso nao dês logout como deve ser a sessão vai continuar ativa, mesmo depois de encerrares o server
    - python client_cli.py prune-sessions   - elimina todas as sessões que nao servem de nada
    - python client_cli.py whoami           - diz a sessão em que estás
    - python client_cli.py active           - parecido ao sessions, mas so te diz as sessões que estão BOAS para se utilizar, o session diz todas
    - python client_cli.py logout           - dar logout como deve ser
